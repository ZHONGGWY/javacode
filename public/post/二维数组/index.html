                int col = 0;
                System.out.println("输入二维数组第"+i+"行，第"+j+"列");
                col = sc.nextInt();
                arr[i][j] = col;
            }
        }
        for (int[] ints : arr) {
            for (int j = 0; j < arr[0].length; j++) {
                System.out.print(ints[j]+"\n");
            }
            System.out.println(" ");
        }
    }

二维数组的使用细节和注意事项

  1. 一维数组的声明方式有：

int[] x / int x[]

  1. 二维数组的声明方式有：

int[][] arr / int[] arr[] / int arr[][]

  1. 二维数组实际上是由多个一维数组组成的，他的各个一维数组的长度可以相同也可以不同。例如：
    arr[][] = {{1,2},{2,3,4}}; arr[][] 是含有两个一维数组的二维数组，arr[0]是含有两个元素的一维数组。这样的数组称为列数不等的二维数组。

  2. 在int[] x,y[] 以下允许通过编译的有那些：
    说明：x是一维数组 ，y是二维数组

    1. x[0] = y 错误 int[][] -> int
    2. y[0] = x 正确 int[] -> int[]
    3. y[0][0] = x 错误
    4. x[0][0] = y 错误
    5. y[0][0] = x[0] 正确int->int
    6. x=y cuowu
