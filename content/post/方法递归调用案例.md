# 方法递归调用
* 递归案例
   1. 打印问题
```
public class Demo1 {
    public static void main(String[] args) {
        T t = new T();
        t.test(4);
    }
}
class T {
    public void test(int n){
        if (n>2){
            test(n-1);
        }
        System.out.println(n);
    }
}
```

   ```
    public int factorial(int n){
        if (n==1) {
            return 1;
        }else {
            return factorial(n-1) * n;
        }
    }
   ```
